static IHostBuilder VaultServiceHostingExtensions.UseVault ( this IHostBuilder  builder)
inlinestatic

Loads configuration from a vault that is configured via environment variables and local files. All placeholders in the configuration are replaced by values from vault. Placeholders look like this: {% <type> <key> %}

Parameters
builderThe host builder
Returns
The same host builder

Only settings configured in ConfigureHostConfiguration have their placeholders replaced. Placeholders in settings loaded during ConfigureAppConfiguration are not replaced.

Exceptions
VaultConfigurationSourceExceptionThe exception thrown if it can't connect to Vault.
